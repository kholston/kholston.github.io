Who I am

Front Stack Web Developer

I am a full stack developer with a focus on front end development. With over a year of self study, I have built up a knowledgebase to create modern websites and web apps. Before I decided to start this journey my web development familiarity only covered basic HTML and CSS. With an interest in maker projects and iot programming I started to research open source projects online. Finding out that wealth of desktop applications built using javascript frameworks intrigued me. As I explored more projects I started to look beyond the individual projects towards the language and architecture underpinning them. Once learning that web development was so much more than my initial thoughts a need to learn more was kindled. I continue to learn more everyday about the many facets of software engineering.
